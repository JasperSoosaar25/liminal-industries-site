# Liminal Industries – Modpack Viewer

🔗 **Live Site:** https://jaspersoosaar25.github.io/liminal-industries-site/

A clean, public viewer for exploring everything inside the **Liminal Industries** modpack — including all mods, plus any custom or modified content.

---

## 📦 What This Site Shows

* Complete mod list (148 entries)
* Simple, scrollable interface
* Highlighted custom content such as:

  * KubeJS scripts (custom recipes / mechanics)
  * Resource pack overrides
  * Patchouli books (if present)

This makes it much easier to understand what’s actually inside the pack without digging through files.

---

## 📁 Project Structure

```
/

```

---

## 🛠️ About the Data

All mod information is stored in `data.json`.

The site reads from this file to display:

* Mod names
* Any notes about modifications
* Custom content indicators

---

## ⚙️ Custom Content

This modpack includes custom behavior through:

* **KubeJS** → used to modify recipes, progression, and gameplay systems
* **Resource packs** → for visual or audio tweaks
* **Patchouli content** → for in-game documentation (if included)

These are separated in the viewer so they’re easy to spot.
